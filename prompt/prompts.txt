ARM Template
------------------
create ARM template for below requirement: Create resource group inb and resource inside it for central region,
two virtual network - ibn-gateway, inb-core and peer them. For ibn-gateway vnet, create 1 subnet and install application gateway. Define NSG for internet incoming request and forward to ui app service running on port 8080. For ibn-core vnet, create 4 subnets: 1. inb-core-int-sbnet with NSG to accept request from application gateway and respond back; ui and core app service call route/installed via this. ui app service internally call to ui-core app service. 2. inb-core-acs-sbnet: core app service to connect privately to azure communication service, 3. inb-core-paygw-sbnet: core app service to connect privately to payment gateway service i.e. paypal, 4. inb-core-blob-sbnet: core app service to connect privately to blob zrs service. configure and create resources i.e. acs, blob zrs, app services for core and ui, and log analytics for all the services and activities.
